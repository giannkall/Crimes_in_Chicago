## Crimes in Chicago



##### An interactive data exploration of Chicago crime patterns across time, location, and offense type to drive proactive public safety decisions.

* ##### Athanasios Sakatziadis

#### Project Description



##### This project delivers a data-driven analysis of city crime patterns across key offense categories in Chicago. Using historical crime data we identify actionable insights that help authorities anticipate, prevent, and respond to crime more effectively. Our goal is to support proactive decision-making that contributes to safer streets and stronger communities.

##### 

##### Using a dataset from Kaggle, we perform data cleaning, exploratory data analysis (EDA), and build an interactive dashboard to visualize hidden patterns

#####
